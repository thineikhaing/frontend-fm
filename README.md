## Running Locally

The application requires Ruby on Rails 6, NodeJs and MongoDB to run.
In order to run this locally, the following is required:

+ Node (version 16.17.0)
+ npm (version 8.19.2)


Running on Frontend
```sh
$ cd frontend-fm
$ npm set audit false
$ npm cache clean --force 
$ npm install
$ npm run start
```
